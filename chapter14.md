## 第14章 重载操作符与转换

* 重载操作符是具有特殊名称的函数： 保留字 `operator`后接需要定义的操作符符号。
* 可以重载的操作符：
| | | | | | | | | | | | |
---|---|---|---|---|---|---|---|---|---|---|
\+ | \- | \* | / | % | ^ | & | \| | ~ | ! | , | = 
< | > | <= | \>= | ++ | = | << | \>> | == | != | && | \|\| 
+= | -= | /= | %= | ^= | &= | \|= | \*= | <<= | \>>= | [] | () 
-> | ->* | new | new [] | delete | delete [] 
---|---|---|---|---|---|---|---|---|---|---|

* 不可以重载的操作符：

| | | |
---|
:: | .* | ?: |
---|

* 通过连接其它合法操作符可以创建新的操作符。
